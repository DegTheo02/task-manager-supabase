# Task Manager (Supabase + Vite React)

## Quickstart

1. Create a Supabase project and run `sql/init.sql` to create the `tasks` table and policies.
2. Set environment variables:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
3. Install & run:
   ```
   npm install
   npm run dev
   ```
4. Deploy to Vercel by pushing to GitHub and importing the repo.

See `sql/init.sql` for the exact SQL to run in Supabase.
